# NSL Company Website

A modern, responsive company website built with Next.js and Bootstrap.

## 🚀 Features

- **Modern Design**: Clean and professional user interface
- **Responsive Layout**: Optimized for all device sizes
- **Interactive Components**:
  - Image carousels
  - Animated client logos
  - Service tabs
  - Contact forms
- **Performance Optimized**: Built with Next.js and Turbopack

## 🛠️ Tech Stack

- [Next.js](https://nextjs.org/) - React framework
- [React](https://reactjs.org/) - UI library
- [Bootstrap](https://getbootstrap.com/) - CSS framework
- [Font Awesome](https://fontawesome.com/) - Icons
- [EmailJS](https://www.emailjs.com/) - Email functionality

## 🏃‍♂️ Getting Started

1. **Clone the repository**

   ```bash
   git clone git@github.com:aboodhijazi55/National-specialized.git
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run development server**

   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) to view the site.
